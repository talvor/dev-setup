# Command Line Tools to Install
# One tool per line, comments start with #
# 
# Format:
#   tool_name                   - Install from package manager default repos
#   tool_name#user/repo         - Install from GitHub tap (user/repo)
#   tool_name#user/repo#url     - Install from custom tap URL
#
# Examples:
#   git                         - Install git from default repos
#   terraform#hashicorp/tap     - Install terraform from hashicorp/tap GitHub repo
#   some-tool#company/tools#https://github.com/company/my-tools.git - Install from custom URL
